Here is a simple web server with Node.js

Run it with:

    node server.js

Test it with:

    curl localhost:8080

It will answer with:

    <!DOCTYPE HTML>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>Hello World</title>
    </head>
    <body>
        Hello World
    </body>
    </html>
